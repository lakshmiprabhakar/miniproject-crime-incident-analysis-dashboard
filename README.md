# Crime Incident Analysis Dashboard
The Crime Incident Analysis Dashboard – 2025 focuses on analyzing crime incidents recorded during the year 2025. The raw dataset was first cleaned and organized using Excel to remove inconsistencies, handle missing values, and structure the data properly for analysis. After preprocessing, the cleaned dataset was imported into Power BI to build an interactive and meaningful dashboard.

TIn this project, I analyzed crime incident data for the year 2025 by transforming raw data into meaningful insights using Excel and Power BI.

I started by cleaning and preprocessing the dataset in Excel, where I handled missing values, removed inconsistencies, and structured the data for accurate analysis. After preparing the dataset, I imported it into Power BI to design an interactive dashboard.

I created multiple visualizations including KPI cards to display total incidents, charts for offense type distribution, monthly crime trends, time shift analysis, and location-based crime patterns. I also developed DAX measures to calculate key metrics such as total incidents, distinct crime categories, district-wise analysis, and percentage contributions.

To enhance interactivity, I implemented slicers that allow users to filter data by district, offense type, time shift, and date. Additionally, I built a drillthrough feature that enables deeper analysis at multiple levels (District → Ward → Neighborhood), providing detailed insights.

I also added navigation buttons to improve user experience and ensure smooth movement between different report pages.

Through this project, I demonstrated my skills in data cleaning, data modeling, DAX calculations, dashboard design, and data storytelling using Power BI, effectively converting raw data into actionable insights.
